## Customize

### 主题

当前有两个可用的主题

- `light`(默认)
- `dark`

```ts filename="app.ts"
...
export const rabbyKit = createModal({
  ...
  theme: "dark",
});
...
```

### 字体

通过在`themeVariables` 改变 `--rk-font-family` 来更换字体

### 自定义变量

默认 `themeVariables` 是未定义的，您可以像这样来修改 `themeVariables`：

```ts filename="app.ts"
...
export const rabbyKit = createModal({
  ...
  theme: "dark",
  themeVariables:{
    "--rk-border-radius":"16px"
  }
});
...
```

The following list shows the theme variables you can override:

| Variable                            |     Description      |   Type   |
| :---------------------------------- | :------------------: | :------: |
| `--rk-font-family`                  |     font family      | `string` |
| `--rk-border-radius`                |    border radius     | `string` |
| `--rk-button-border-radius`         | button border radius | `string` |
| `--rk-primary-button-color`         |     button color     | `string` |
| `--rk-primary-button-border`        |  button background   | `string` |
| `--rk-primary-button-border-radius` |  button background   | `string` |
| `--rk-primary-button-bg`            |  button background   | `string` |
| `--rk-primary-button-hover-color`   |     button color     | `string` |
| `--rk-primary-button-hover-bg`      |  button background   | `string` |
| `--rk-primary-button-hover-border`  |  button background   | `string` |
