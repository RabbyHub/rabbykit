# JavaScript and Rabbykit

## Install

<Steps>
### Install

Install ConnectKit and its peer dependencies

```sh npm2yarn
npm i @rabby-wallet/rabbykit @wagmi/core viem
```

### 使用 Rabbykit 和 wagmi

first setup `wagmi`，then setup `rabbykit`

```ts filename="app.ts"
import { configureChains, createConfig, mainnet } from "@wagmi/core";
import { mainnet, arbitrum, bsc, optimism, polygon } from "@wagmi/core/chains";
import { createModal } from "@rabby-wallet/rabbykit";

const { chains, publicClient, webSocketPublicClient } = configureChains(
  [mainnet, arbitrum, bsc, optimism, polygon],
  [
    alchemyProvider({ apiKey: "yourAlchemyApiKey" }),
    infuraProvider({ apiKey: "yourInfuraApiKey" }),
    publicProvider(),
  ]
);

const config = createConfig({
  autoConnect: true,
  publicClient,
  webSocketPublicClient,
});

export const rabbyKit = createModal({
  chains,
  wagmi: config,
  projectId: "yourProjectId",
  appName: "RabbyKit",
});
rabbykit.open();

console.log("current rabbykit modal open status:", rabbyKit.getOpenState());

rabbykit.close();
```

</Steps>
